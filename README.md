# Left-sided Infective Endocarditis Risk Prediction Model

This repository provides an online deployment framework for a CatBoost-based prediction model for left-sided infective endocarditis scheduled to undergo surgery.

The model is designed to estimate the probability of 30-day adverse outcomes after surgery in surgical candidates with left-sided infective endocarditis and to classify patients into high-risk or low-risk groups according to a predefined cutoff.
